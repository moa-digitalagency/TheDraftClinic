## Processus de commande

### Etape 1 : Soumission

Le client cree un compte et soumet sa demande avec :
- Type de service souhaite
- Titre et description du projet
- Volume (mots ou pages)
- Date limite
- Documents de reference

### Etape 2 : Evaluation

L'equipe analyse la demande pour :
- Faisabilite du projet
- Estimation du travail
- Identification des besoins specifiques

### Etape 3 : Devis

Un devis detaille est envoye avec :
- Montant total
- Acompte requis (generalement 50%)
- Delai de realisation

### Etape 4 : Acceptation et paiement

Le client :
- Accepte le devis
- Effectue le paiement de l'acompte
- Soumet la preuve de paiement

### Etape 5 : Realisation

Apres verification du paiement :
- Le travail demarre
- Le client suit la progression en ligne
- Des echanges sont possibles via les commentaires

### Etape 6 : Livraison

Le travail termine est uploade sur la plateforme :
- Le client telecharge les fichiers
- Il peut demander des revisions si necessaire

### Etape 7 : Revision

Si des modifications sont necessaires :
- Le client decrit les changements souhaites
- L'equipe effectue les corrections
- Une nouvelle version est livree
