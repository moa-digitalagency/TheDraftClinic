# Guide Utilisateur TheDraftClinic

Guide complet pour les clients de la plateforme de services de redaction academique.

---

## Presentation

TheDraftClinic est une plateforme de services de redaction academique destinee aux chercheurs, doctorants et etudiants. Elle permet de soumettre des projets de redaction, recevoir des devis, suivre l'avancement des travaux et telecharger les livrables.

---

## Inscription

### Creer un compte

1. Rendez-vous sur la page d'accueil
2. Cliquez sur "Sign up" ou "S'inscrire"
3. Remplissez le formulaire :
   - Email (sera utilise pour la connexion)
   - Mot de passe (8 caracteres minimum)
   - Prenom et nom
   - Telephone (optionnel)
   - Etablissement (universite, ecole)
   - Niveau academique (licence, master, doctorat)
   - Domaine d'etude
4. Validez votre inscription
5. Vous pouvez maintenant vous connecter

### Informations demandees

| Champ | Obligatoire | Utilisation |
|-------|-------------|-------------|
| Email | Oui | Connexion et contact |
| Mot de passe | Oui | Securite du compte |
| Prenom/Nom | Oui | Identification |
| Telephone | Non | Contact rapide |
| Etablissement | Non | Contexte du projet |
| Niveau academique | Non | Adapter le service |
| Domaine d'etude | Non | Expertise appropriee |

---

## Connexion

1. Cliquez sur "Login" ou "Connexion"
2. Entrez votre email et mot de passe
3. Cochez "Se souvenir de moi" si vous utilisez un appareil personnel
4. Cliquez sur "Se connecter"

Si vous avez oublie votre mot de passe, contactez le support.

---

## Tableau de bord

Apres connexion, vous accedez a votre tableau de bord qui affiche :

### Statistiques

- Total de vos demandes
- Demandes en cours de traitement
- Demandes en attente (devis, paiement)
- Demandes terminees

### Liste des demandes

Toutes vos demandes sont listees avec :
- Titre du projet
- Type de service
- Statut actuel
- Progression (barre visuelle)
- Date de soumission

---

## Soumettre une demande

### Etape 1 : Nouvelle demande

1. Cliquez sur "Nouvelle demande" depuis le tableau de bord
2. Remplissez le formulaire

### Etape 2 : Informations du projet

| Champ | Description |
|-------|-------------|
| Type de service | Selectionnez parmi la liste (these, memoire, article...) |
| Titre | Titre de votre projet |
| Description | Expliquez en detail vos attentes |
| Nombre de mots | Estimation du volume souhaite |
| Nombre de pages | Alternative au nombre de mots |
| Date limite | Quand avez-vous besoin du travail |
| Niveau d'urgence | Standard, Express 24h, Express 5h |
| Informations additionnelles | Consignes specifiques |

### Etape 3 : Documents

Vous pouvez joindre des documents de reference :
- Plan, structure souhaitee
- Sources bibliographiques
- Documents existants a retravailler
- Consignes de votre etablissement

Formats acceptes : PDF, Word (doc/docx), texte, images.

### Etape 4 : Soumission

Cliquez sur "Soumettre la demande". Vous recevrez une confirmation et pourrez suivre l'evolution.

## Cycle de vie d'une demande

### 1. Demande soumise

Votre demande est recue et en attente d'examen par l'equipe.

### 2. Examen

Un expert analyse votre demande pour evaluer le travail necessaire.

### 3. Devis envoye

Vous recevez une proposition avec :
- Montant total du projet
- Montant de l'acompte (generalement 50%)
- Message explicatif

### 4. Acceptation du devis

Si le devis vous convient :
1. Consultez les details
2. Cliquez sur "Accepter le devis"

### 5. Paiement de l'acompte

Apres acceptation du devis :
1. Effectuez le paiement selon la methode indiquee
2. Soumettez la preuve de paiement :
   - Montant paye
   - Methode (virement, PayPal, carte)
   - Reference de transaction
   - Capture d'ecran ou recu

### 6. Travail en cours

Une fois le paiement verifie, le travail commence. Vous pouvez suivre la progression (pourcentage) sur la page de votre demande.

### 7. Livraison

Le travail termine est uploade. Vous pouvez telecharger les fichiers.

### 8. Revisions

Si des modifications sont necessaires :
1. Decrivez les changements souhaites
2. Joignez eventuellement des documents
3. Soumettez la demande de revision

## Suivi d'une demande

### Page de detail

Chaque demande affiche :
- Informations du projet
- Statut actuel
- Progression
- Documents joints (vos uploads et livrables)
- Historique des paiements
- Journal d'activite

### Historique d'activite

Le journal montre toutes les actions :
- Commentaires de l'equipe
- Livraisons de fichiers
- Changements de statut
- Vos telechargements

### Communication

Vous pouvez ajouter des commentaires pour communiquer avec l'equipe directement sur la page de la demande.

---

## Telechargement des livrables

1. Ouvrez la page de votre demande
2. Reperer les documents de type "Livrable"
3. Cliquez sur le bouton de telechargement
4. Le fichier est enregistre sur votre appareil

Chaque telechargement est enregistre dans l'historique d'activite.

---

## Extension de delai

Si l'equipe a besoin de plus de temps, vous recevrez une demande d'extension :

1. Consultez la nouvelle date proposee
2. Lisez la raison de la demande
3. Choisissez "Approuver" ou "Refuser"
4. Ajoutez un message si necessaire

---

## Profil utilisateur

### Modifier vos informations

1. Cliquez sur "Profil" dans le menu
2. Modifiez les champs souhaites
3. Cliquez sur "Enregistrer"

### Informations modifiables

- Prenom et nom
- Telephone
- Etablissement
- Niveau academique
- Domaine d'etude

L'email ne peut pas etre modifie (contactez le support si necessaire).

---

## Changement de langue

La plateforme est disponible en francais et anglais.

1. Localisez le selecteur de langue (en bas de page ou dans le menu)
2. Choisissez votre langue preferee
3. L'interface se met a jour immediatement

## Conseils

1. Soyez precis dans votre description pour eviter les allers-retours
2. Joignez tous les documents utiles des le depart
3. Indiquez une date limite realiste
4. Consultez regulierement l'avancement
5. Utilisez les commentaires pour poser des questions
6. Telechargez vos livrables rapidement

---

## Support

Pour toute question ou probleme :
- Utilisez le formulaire de contact sur le site
- Les commentaires sur vos demandes sont aussi lus par l'equipe
